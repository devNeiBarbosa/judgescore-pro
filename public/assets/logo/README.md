# Assets oficiais — JUDGESCORE PRO

Diretório oficial de identidade visual.

## Arquivos e uso

- `icon-symbol.png`
  - Fallback global (favicon, loaders, PDF, e-mail, estados vazios).
- `icon-main.png`
  - Ícone para UI em fundo escuro.
- `icon-light.png`
  - Ícone para UI em fundo claro.
- `logo-primary-dark.jpg`
  - Logo principal para fundo escuro.
- `logo-primary-light.jpg`
  - Logo principal para fundo claro.
- `logo-transparent-dark.png`
  - Logo transparente para fundos claros.
- `logo-transparent-light.png`
  - Logo transparente para fundos escuros.
- `logo-vertical-dark.jpg`
  - Logo vertical para login/splash/telas institucionais.

## Regras de marca

- Não inverter cores via CSS.
- Não aplicar sombras/efeitos no logo.
- Não distorcer proporção (sempre `width: auto` com altura fixa).
- Sempre usar a versão correta de acordo com o fundo.
